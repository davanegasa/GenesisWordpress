# Directorio `admin/`

Aquí se encuentra el código del dashboard administrativo de Genesis.

- `dashboard.php`: Página de administración para visualizar logs y futuras herramientas administrativas.
- `index.php`: Archivo de seguridad para evitar el listado de directorios.

Este dashboard permite a los administradores ver los logs del sistema y servirá como base para futuras funcionalidades de gestión y monitoreo.
